## 📝 Documentation Standards

All documents in this directory follow these conventions:

- **Format**: Markdown (`.md`)
- **Headings**: H1 for title, H2 for major sections, H3 for sub-sections
- **Tables**: used for structured reference material
- **Code blocks**: language-tagged fenced blocks (e.g., ` ```bash `)
- **Links**: relative paths within `docs/`, absolute URLs for external resources
- **Last Updated**: noted at the bottom of each document
- **Review cycle**: quarterly or on significant code changes

To suggest improvements, open an issue with the label `documentation`.
